```
Job Search Strategy:

   - Apply to 5-10 companies per week

   - Study coding questions
   - Practice system design
   - Research companies
   - Mock interviews

   - Get written offer
   - Research salary
   - Negotiate benefits
   - Accept and celebrate!
```

## 3ï¸âƒ£ Salary Negotiation

### Salary Research

```
Salary Research Sources:
- Levels.fyi (compare by company)
- Glassdoor (company reviews)
- PayScale (salary data)
- BLS (Bureau of Labor Statistics)
- Blind (anonymous posts)

Factors Affecting Salary:
- Location (SF, NYC, Seattle highest)
- Company size (FAANG pays premium)
- Years of experience
- Specialization
- Negotiation skills

Example Ranges (USA):
Junior: $60K-$90K
Mid: $90K-$150K
Senior: $150K-$250K+
Staff: $200K-$400K+

Total Compensation:
- Base salary: 60-70%
- Bonus: 10-30%
- Stock/equity: 10-20%
- Benefits (401k, health, etc.)

Negotiation Strategy:
1. Research market rate
2. Get offer in writing
3. Ask for time to consider
4. Counter politely
5. Negotiate total package
6. Focus on value provided
```

### Negotiation Script

```
Template for Negotiations:

1. Thank them for offer:
"Thank you for the offer. I'm excited about the opportunity."

2. Express enthusiasm:
"I'm confident I can contribute significantly to the team."

3. Make case for higher salary:
"Based on my research and the value I'll provide, I was 
expecting salary in the range of $X-$Y. Can we discuss?"

4. Listen and respond:
- If refused: "Are there other forms of compensation?"
- If accepted: "Great! Can we also discuss..."
- If counter-offered: Accept or negotiate further

5. Negotiate benefits:
- Flexible hours
- Remote work
- Professional development budget
- Extra vacation
- Sign-on bonus

6. Finalize:
"These terms work well for me. When can I start?"

Key Points:
- Always get offer in writing first
- Research market rate beforehand
- Be professional and grateful
- Don't be afraid to negotiate
- Everything is negotiable
```

## Docker Deployment

### Backend Dockerfile

```dockerfile
# backend/Dockerfile
FROM node:18-alpine

WORKDIR /app

# Copy package files
COPY package*.json ./

# Install dependencies
RUN npm ci --only=production

# Copy source code
COPY src ./src
COPY migrations ./migrations

# Health check
HEALTHCHECK --interval=30s --timeout=10s --start-period=5s --retries=3 \
  CMD node healthcheck.js

# Expose port
EXPOSE 5000

# Start application
CMD ["node", "src/index.js"]
```

### Frontend Dockerfile

```dockerfile
# frontend/Dockerfile
FROM node:18-alpine as builder

WORKDIR /app

COPY package*.json ./
RUN npm ci

COPY . .
RUN npm run build

# Production stage
FROM nginx:alpine

COPY --from=builder /app/dist /usr/share/nginx/html
COPY nginx.conf /etc/nginx/conf.d/default.conf

EXPOSE 80

CMD ["nginx", "-g", "daemon off;"]
```

## Monitoring & Observability

### Health Checks

```javascript
// src/health.js
const express = require('express');
const db = require('./config/database');
const redis = require('./config/redis');

const app = express();

app.get('/health', (req, res) => {
  const health = {
    status: 'UP',
    timestamp: new Date(),
    uptime: process.uptime()
  };
  res.json(health);
});

app.get('/health/live', (req, res) => {
  // Kubernetes liveness probe
  res.json({ status: 'ALIVE' });
});

app.get('/health/ready', async (req, res) => {
  // Kubernetes readiness probe
  try {
    await db.query('SELECT 1');
    await redis.ping();
    res.json({ status: 'READY' });
  } catch (err) {
    res.status(503).json({ status: 'NOT_READY', error: err.message });
  }
});

app.listen(3000);
```
